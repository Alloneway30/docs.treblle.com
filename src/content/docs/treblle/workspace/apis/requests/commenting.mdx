Treblle users can now collaborate like never before. In addition to collaborative platform features such as Workspaces, Sharing, Roles, and Catalog, which allow users to work more effortlessly together, users can now have live conversations within the platform.

## Requests

Users are able to comment or have live conversations on individual Requests. To do this, simply click on the chat bubble icon at the top right of an individual request screen to open the Comments sidebar. You are able to tag people who are members of the API by using @username.
